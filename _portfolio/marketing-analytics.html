Introduction

This project simulates a retail business that is facing a drop in customer engagement and conversion rates, despite the launching of several new marketing campaigns.

These are the main key points the company wishes to solve:

  • Reduced Customer Feedback: The number of feedback entries and engagement with the site has declined
  • High Marketing Expenses: Significant investments in marketing campaigns don’t have the expected returns
  • Need for Customer Feedback Analysis: Understanding customer opinions about the products and services is crucial for improving engagement and conversions

Data Source

The original dataset and business problem were created by Ali Ahmad, which inspired this project.

The database contains the following tables:

  • Customer Journey Table: Tracks customer movements through the website to analyze the conversion funnel
  • Engagement Data Table: Measures engagement with different types of content
  • Customer Reviews Table: Analyzes customer feedback to identify common themes
  • Customer Table: Provides additional information about customers
  • Geography Table: Provides additional geographic information about customers
  • Product Table: Provides additional information about products

Tools

  • SQL: Used to clean and transform the data loaded in SQL Server
  • Python: Used to calculate sentiment scores based on customer feedback and to create sentiment categories
  • Power BI: Used to model and create the data visualizations
